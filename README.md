## 🚀 Instalacija

1. **Clone repositorij:**
```bash
git clone [repository-url]
cd zdravlje-i-hrana
```

2. **Upload u Shopify:**
   - Zip sve fajlove
   - Upload kroz Shopify Admin → Themes
   - Ili koristi Shopify CLI

3. **Konfiguracija:**
   - Postaviti slike u `/assets/`
   - Konfigurirati collection-e
   - Udesiti settings u Theme Editor
